# 1\. Introducción

# El Modelo de Tarifas Farmacéuticas – Vigencia 2026 es un aplicativo web diseñado para apoyar a la ESE HUHMP en la estructuración, análisis y definición de tarifas de venta de medicamentos y dispositivos médicos, a partir de información histórica de costos, precios y volúmenes, incorporando criterios técnicos, financieros y normativos.

# El aplicativo permite: - Cargar archivos maestros en Excel (multilibro). - Analizar inventarios mediante metodología Pareto. - Definir y validar estructuras de costos indirectos. - Caracterizar el portafolio por categorías estratégicas. - Modelar tarifas 2026 por libro, con categorización automática. - Generar un Acta PDF oficial, lista para comité o mesa técnica.

# Este manual está dirigido a usuarios finales (financieros, costos, farmacia, planeación y directivos), y describe el uso funcional del sistema, tal como se entrega al cliente final.

# 

# 2\. Requisitos Técnicos

# •	Navegador moderno (Google Chrome, Microsoft Edge o Firefox actualizado).

# •	Archivo maestro en formato Excel (.xlsx / .xlsm).

# •	Archivo de estructura de costos indirectos en Excel.

# •	Conexión a internet (opcional). El sistema puede operar en modo offline, pero sin sincronización.

# 

# 3\. Estructura General del Aplicativo

# El aplicativo se organiza en siete (VII) secciones funcionales, más un módulo transversal de firmas y conclusiones. Cada sección responde a una fase del proceso tarifario.

# 

# 4\. Descripción de Secciones

# I. Resumen de Productos por Tipo

# Objetivo: Cargar y consolidar el archivo maestro institucional.

# Funcionamiento: - El usuario carga un archivo Excel multilibro. - Cada hoja del archivo se interpreta como un Libro (ej. Medicamentos Generales, Regulados, Dispositivos, etc.). - El sistema crea botones dinámicos por cada libro identificado. - Al seleccionar un libro, se muestra un panel resumen con: - Costo total del inventario. - Valor de venta proyectado. - Variación entre costo y venta.

# Información clave: - Esta sección es obligatoria para habilitar el resto del aplicativo. - La correcta estructura de encabezados en el Excel es fundamental.

# 

# II. Análisis de Pareto

# Objetivo: Identificar los ítems críticos del inventario bajo el principio 80/20.

# Funcionalidades: - Análisis consolidado o por libro. - Gráfico de barras (conteo por clasificación A, B, C). - Gráfico de torta con distribución porcentual.

# Interpretación: - Tipo A: ~80% del valor del inventario. - Tipo B: ~15% del valor. - Tipo C: ~5% del valor.

# Esta sección apoya la priorización estratégica del portafolio.

# 

# III. Detalle de Ítems Farmacéuticos

# Objetivo: Visualizar el inventario completo con capacidad de búsqueda y validación.

# Funcionalidades: - Filtro por libro. - Búsqueda por código o descripción. - Visualización de: - Precio 2025. - Unidades. - Valor máximo de compra. - Estado, observaciones y Pareto.

# Uso recomendado: - Validar datos inconsistentes antes de avanzar al análisis tarifario.

# 

# IV. Estructura de Costos Totales

# Objetivo: Definir los costos indirectos que afectan el costo total por ítem.

# Funcionamiento: - Carga de archivo Excel de costos indirectos. - Clasificación por: - Costos logísticos. - Costos administrativos. - Gastos fiscales y tributarios. - Incremento estimado al precio de compra siguiente vigencia.

# El sistema calcula automáticamente el porcentaje total de costos indirectos, que se aplica a cada ítem.

# 

# V. Caracterización de Bienes

# Objetivo: Clasificar estratégicamente el portafolio institucional.

# Categorías: - BLACK - ELITE - PLATINO - ORO - PLATA - CLÁSICOS - REGULADOS - PRODUCTOS SIN FRECUENCIA DE USO

# Criterio técnico: - La categorización se basa en el margen histórico (Precio 2025 vs. Costo Total).

# Funcionalidades clave: - Incremento general 2026. - Filtros por categoría. - Gráficos interactivos (torta y barras). - Descarga de Excel completo con la caracterización.

# 

# VI. Firmas

# Objetivo: Formalizar el documento como acta institucional.

# Funcionamiento: - Registro dinámico de firmantes (Elaboró, Revisó, Aprobó, etc.). - Esta información se incorpora automáticamente en el PDF final.

# 

# Conclusiones y Recomendaciones

# Objetivo: Registrar los acuerdos de la mesa técnica o comité tarifario.

# Características: - Campo de texto libre. - El contenido se incorpora en el acta PDF, debajo de las firmas. - Si se deja vacío, el PDF genera espacio para diligenciamiento manual.

# 

# VII. Análisis de Tarifas por Libro (Detallado)

# Objetivo: Definir las tarifas de venta 2026 por libro y por categoría.

# Funcionalidades principales: - Análisis independiente por cada libro. - Buscador individual por libro. - Filtros por categoría y observaciones. - Configuración de incrementos (%) por categoría.

# Cálculos automáticos: - Costo total = Precio de compra + costos indirectos. - Tarifa 2026 según categoría. - Margen ($). - Variación % = (Margen / Costo Total) × 100.

# Categorización automática: - El sistema reclasifica los ítems según el margen. - Los ítems regulados mantienen su condición especial.

# Acciones clave: - Guardar: obligatorio para que la información quede incluida en el acta. - Descargar Excel: genera archivo por libro con el detalle del análisis.

# 

# 5\. Generación del Acta PDF

# El botón “Descargar Acta” genera un documento PDF oficial que incluye: - Resúmenes. - Gráficos. - Tablas consolidadas. - Firmas. - Conclusiones y recomendaciones.

# Este documento está diseñado para presentación ante: - Comité de tarifas. - Subgerencia administrativa y financiera. - Gerencia.

# 

# 6\. Consideraciones Finales

# •	Siempre verificar la correcta carga de archivos antes de generar el acta.

# •	Guardar cada libro en la Sección VII antes de descargar el PDF.

# •	El sistema permite trazabilidad y control técnico del proceso tarifario.
