Manual de Uso - Modelo Tarifario

Bienvenido. Este aplicativo permite cargar datos maestros, analizar el comportamiento de costos, clasificar el portafolio y definir tarifas 2026.

1 Carga de Datos Iniciales

  • Archivo Maestro: En la Sección I, cargue su Excel (.xlsx). Cada hoja se interpreta como un "Libro" (Medicamentos, Dispositivos, etc.).

2 Análisis de la Información

Visualice Pareto (Sección II) y la Caracterización Global (Sección V).

3 Definición de Tarifas (Sección VII)

Aquí se gestiona cada libro por separado. Use el gráfico de torta para ver la distribución y la tabla para el detalle.

  • Búsqueda Individual: Use el cuadro de búsqueda en la cabecera de cada tabla para filtrar ítems por nombre o código.
  • Auto-Categorización: El sistema asigna automáticamente categorías (Black, Elite, Platino...) según el margen de rentabilidad, excepto para regulados.
  • Incrementos por Categoría: Ajuste el % de incremento para cada grupo.
  • Guardar: Es OBLIGATORIO hacer clic en "Guardar" para incluir la configuración en el PDF.
